# JSP-BookStore
JSP - 책 쇼핑몰

### 주요기능

+ 관리자
  + 책 관리(삭제,수정,추가)
  + 사용자가 구매한 목록 
  
+ 사용자
  + 장바구니(담기,삭제)
  + 구매(수량 선택 가능)
